# Information Security Policy

## Access Control

All corporate systems require multi-factor authentication. Passwords must be
at least 14 characters and rotate every 180 days. Shared accounts are
prohibited without a documented exception approved by the security team.

## VPN and Remote Access

Remote access to internal systems is allowed only through the corporate VPN.
VPN client certificates rotate every 90 days automatically. Split tunneling
is disabled on all managed devices.

## Incident Response

Suspected security incidents must be reported to the security operations
center within one hour of discovery. The on-call responder triages incidents
using a four-level severity scale. Severity 1 incidents trigger an executive
notification within four hours.

## Data Classification

Data is classified into public, internal, confidential and restricted tiers.
Restricted data may not leave the production environment and access requires
quarterly recertification by the data owner.
